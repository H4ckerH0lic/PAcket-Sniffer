Basic Terms

ARP = Address Resolution Protocol
ARP-Request = ARP broadcasts a request packet to all the machines on the LAN and asks if any of the machines are using that particular IP address.
ARP-Response = It is the MAC address response that the source receives from the destination which aids in further communication of the data.
IP-ADDRESS = a unique numerical identifier for every device or network that connects to the internet.
MAC-ADDRESS = a unique, 12-character alphanumeric attribute that is used to identify individual electronic devices on a network.
Gateway = A gateway is a network node used in telecommunications that connects two networks with different transmission protocols together.
Router = A router is a networking layer system used to manage and forward data packets to devices network while a gateway is simply a hardware that acts as a gate between the networks.
